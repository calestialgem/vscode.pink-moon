# Pink Moon Theme

A medium-dark theme with seal blue and salmon red. Colors are taken from the
[Pink Moon Theme](https://github.com/sts10/vim-pink-moon) for VIM/NeoVIM.

**Only tested for C, Markdown and JSON!**
